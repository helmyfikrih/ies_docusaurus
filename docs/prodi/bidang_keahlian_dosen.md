## Menambah Keahlian Dosen

1.  Membuka halaman list keahlian dosen pada sub menu **Pengaturan** -> **Master Data** -> **Bidang Keahlian Dosen**.
2.  Ketik **nama dosen** yang ingin di tambahkan keahliannya.
3.  Input **bidang keahlian**.
4.  Klik **Save**.
